Twig of Life

A visualization of a small part of the great tree of life. This web app uses D3 and Angular to create an interactive radial dendrogram.

Seismic

Creates a map of recent significant earthquakes using earthquake data from the U.S. Geological Survey. Technologies used: SVG, jQuery, and plain CSS.

Dander

A dog discovery app that helps you find dogs near your area that are available for adoption. You can curate a list of dog's you are interested in!

Dander consists of an Express API server which is consumed by a front-end static site. Authentication is handled via JSON Web Tokens and Passport.js.
